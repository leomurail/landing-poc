# Landing page

## Getting Started

### Start install

```bash
$ npm install
```

### Run start

```bash
$ npm run dev
```

## Deployement

### Run build

```bash
$ npm run build
```

### Run preview

```bash
$ npm run preview
```

## Tests

### Run ESLint tests

```bash
$ npm run lint
```

Les fichiers construit vont dans /dist
